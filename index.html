Weather FORE-cast

Web application I built using HTML5, Bootstrap, JavaScript & jQuery. It allows the user to search for a golf course where they will be playing in the near future, input the time and date they will be playing and the app returns the weather forecast for that time, date and location. Road Trippin'

I built this application using React, CSS, NodeJS, Express & PostgreSQL. I also had a little help from React Router, FontAwesome, date-fns, AWS S3, multer, knex, xss and jsonwebtoken. It allows the user to register an account, log in, view images uploaded by themselves or other users, comment on and rate images, search by location, as well as upload, edit and delete images they've added. I built it so the user can upload an image to an AWS S3 bucket and then the url for that location is stored in the PostgreSQL database for retrieval.

GeekBox

This application was created to aggregate educational resources available on the web by category. It allows the user to optionally register an account and securely sign in. Once authenticated the user can add categories and resources, as well as comment on and rate individual resources. If user should decline to sign in they can still browse the categories, resources and comments added by other users.

The technologies I used to build this were React, React Router, CSS3, FontAwesome, date-fns, NodeJS, Express, bcrypt, dotenv, helmet, jsonwebtoken, knex, morgan, treeize, xss and PostgreSQL.
